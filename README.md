# hello_vue

## Project setup

node.js 설치

    - https://nodejs.org/en/download/
    - LTS 버전 추천 (뷰와 버전호환이 안될 수 있음)

npm 사용한 Vue.js 설치

    - npm install vue@next
    - npm install -D @vue/compiler-sfc

    - CLI 설치
        - npm install -g @vue/cli

프로젝트 생서

    - npm init vite-app <project-name>
    - cd <project-name>
    - npm install
    - npm run dev

프로젝트로에서 vue 업데이트

    - vue upgrade --next
